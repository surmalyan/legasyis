---
name: Memory Circle purpose
description: Core emotional purpose of Memory Circle — closing the pain of loss by collecting new stories from friends/family
type: feature
---
Memory Circle exists to ease the pain of losing a loved one by giving the bereaved a way to discover NEW stories about them.

Core flow:
- Friends and relatives contribute memories (text, voice, photos)
- The bereaved (and others) can read or listen to those stories
- The deceased's voice is never imitated — only real memories from real people

Design implications:
- Tone: warm, empathic, never cheerful or gamified
- Prioritize voice playback (hearing real voices of contributors telling stories)
- Make it easy for non-tech contributors to add a memory in <1 minute
- Surface "new stories since last visit" prominently for the bereaved
- Never frame contributing as a "task" — frame as "share a memory"
